MODULO DE CLIENTES

Los Clientes a traves de este modulo podran realizar la consulta de productos,tiendas y hacer los pedidos respectivos.

MODULO TIENDAS

Los propietarios de las tiendas a través de este módulo podrán realizar la consulta de productos que ofrecen los agricultores y hacer los pedidos.

MODULO DE COSECHA

A través de este modulo se podrán consultar los tiempos de cosecha para cada uno de los productos cultivados en la región.

MODULO PRODUCTOS

A través de este módulo se podrán consultar todos los productos; precio, y las tiendas que los ofrece.

MODULO MUNICIPIOS

A través de este módulo se podrán consultar todos los municipios que están cubiertos por el sistema de información.

MODULO TRANSPORTE

A través de este módulo se podrá consultar acerca de las tarifas de transporte para desplazamiento de los productos

MODULO CULTIVOS

A través de este módulo se podrán consultar todos los cultivos de productos Hortofruticolas de la Región.

MODULO TIENDA INSUMOS

A través de este módulo se podrán consultar las diferentes tiendas de insumos agrícolas de la región.

MODULO AGRICULTORES

Los Agricultores a través de este módulo podrán contactar a tiendas de barrio y tiendas de insumos agricolas.
